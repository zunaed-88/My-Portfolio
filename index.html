PET IMAGE TRANSLATION WITH PIX2PIX

A GAN-based model for translating PET Non attenuated corrected images to PET CT based attenuated corrected images.This model generates a good quality image with SSIM @ 95 %, PSNR@ 31%, and VIF @91% with low reconstruction errors .

Python TensorFlow OpenCV Nibabel TensorBoard NumPy Matplotlib Scikit-Learn Pydicom

Transfer learning with Auto-encoders and t-SNE for Cross-Dataset Generalization

This project introduces an autoencoder-based transfer learning approach that enhances deep learning performance in medical imaging. By reusing learned feature representations, it improves model accuracy and efficiency across NIH Chest X-Ray Dataset and COVID-19 Radiography Dataset.

Python TensorFlow Keras Auto-encoders Scikit-Learn Numpy Matplotlib kaggle t-SNE Transfer Learning
GitHub

MRI Segmentation for Tumors

In this project I built a multi-class model for segmenting an MRI image.This model identify 3 different abnormalities in each image: edemas, non-enhancing tumors, and enhancing tumors

Python TensorFlow NumPy Keras Nibabel Pandas Git

Concrete Crack Detector

In this project we trained an AI model that classify concrete crack images using deep learning. In this project I used transfer learning on VGG16 and ResNet50 pre-trained model for on concrete crack data for the detection of cracks.Both models VGG16 and ResNet50 yeilds accuracy of 98.6% and 96.7% respectively

Python Pandas TensorFlow Keras Scikit-Learn Numpy Matplotlib Seaborn
